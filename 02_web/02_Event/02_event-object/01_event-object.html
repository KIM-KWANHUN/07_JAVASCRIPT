01. 이벤트 객체

    
이벤트 객체란?...

        이벤트 발생 시 이벤트와 관련 된 객체가 동적으로 생성이 됨.
        생성이 된 이벤트 객체는 이벤트 핸들러의 첫 번째 인수로 전달이 된다.

아무곳이나 클릭해보세요!!! 좌표를 알려드릴께요!!

        위에서 한 것은 프로퍼티 방식으로 이벤트 등록
        이제 attribute 방식으로 이벤트 등록 시 유의점
        attribute 방식으로 이벤트를 등록했다면 전달인자는
        반드시 event 라는 이름으로 작성해야 한다.
이 영역 내부를 클릭하시면 좌표를 알려드리겠습니다!!
