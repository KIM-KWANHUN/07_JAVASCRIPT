01. open, alert, confirm, prompt 메소드

        
window 객체란?...

        자바 스크립트 언어는 브라우저 상에서 동작하는 언어이다.
        자바 스크립트가 web 상에서 동작을 할 때
        사용할 수 있는 기능이 있다.
        web 상에서 가장 최상위 객체는 window 객체 이다.
        window
            - DOM (document 등등)
            - BOM (location, navigator 등등)
            - JavaScript (Array, Funtion, Object 등등)
        최상단의 window 객체는 브라우저 창을 대변하고
        브라우저를 제어할 수 있는 메소드를 제공한다.
        window 는 전역 객체이기 때문에 생략이 가능하다.
        ex) window.alert() -> alert() window는 생략 가능하다.
    
        
open 이란?...

        표현식 : window.open(url, name(팝업의 이름 즉 식별할 수 있는 이름), params(쉼표를 통해 여러가지를 전달 할 수 있다.))
        메소드로 새로운 창을 열 수 있다.
    
        
alert 란?...

        window.alert(message) 메소드는 확인 버튼을 가지며
        메세지를 지정할 수 있는 경고 상자를 띄워준다.
    
        
confirm 이란?...

        window.confirm(message) 메소드는 확인과 취소 두 버튼을
        가지고 있으며 메세지를 지정할 수 있게 된다.
        반환값이 존재하며, 확인 누르면 true, 취소 or ESC 누르면
        false 를 반환한다.
    
        
propmt 란?...

        window.prompt(message, default값(처음값))
        message : 사용자가 어떤 값을 입력해야 되는 지 안내문구
        default : 텍스트 입력 필드에 기본 값(문자열)
        반환값은 사용자 입력 값이며, 취소를 누르거나
        ESC 키를 누르게 되면 NULL 값 리턴
