05. 노드 복사, 교체, 삭제

노드 복사란?...

        cloneNode([deep : true | false])
        노드의 사본을 생성하여 반환을 해준다.
        매개변수에 true 를 전달하면 노드를 깊은 복사하여
        모든 자손 노드를 포함 된 사본을 반환해주고
        false 를 전달하면 얕은 복사를 하여 노드 자신만의 사본을 생성한다.
