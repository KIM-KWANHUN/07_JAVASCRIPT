01. attribute(속성)

어트리뷰트란?...

        Element(태그) 들은 attribute 라고 하는 프로퍼티가 존재한다.
        ex)p class ="" class같은걸 의미함

get, set 사용

   
        Element 의 getAttribute / setAttribute 메소드를 사용하면
        attributes 객체를 사용하지 않고 접근을 할 수 있다.
    
        Element 가 제공하는
        hasAttribute(속성명) : 인자로 전달한 어트리뷰트 존재 여부
        removeAttribute(속성명) : 인자로 전달한 어트리뷰트 삭제
